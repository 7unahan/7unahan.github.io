RGB-D Digital Twin Dataset for Education

Analyzing student attention with Digital Twin and RGB-D technology

GitHub Paper

Project Overview

The concept of a Digital Twin originated in 1997, but its foundations date back to NASA’s 1960s simulations. It gained renewed interest post-COVID, especially in educational applications. This project proposes a Digital Twin dataset for classrooms using the Intel RealSense D435i RGB-D camera, enabling gaze tracking analysis in a 3D virtual classroom environment.

Technology Stack

  • Intel RealSense D435i RGB-D Camera
  • Point cloud processing via MeshLab
  • Unity 3D for scene creation and animation
  • Gaze line dataset generation for attention tracking
